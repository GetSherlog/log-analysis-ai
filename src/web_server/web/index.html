LogAI-CPP API Server

Welcome to the LogAI-CPP API Server. This server provides a RESTful API for log processing, analysis, and anomaly detection using C++ high-performance implementation.

API Status

GET

/health

Check if the API server is running properly.

Response:

{
  "status": "ok",
  "version": "0.1.0"
}
GET

/api

Get information about the API.

Response:

{
  "name": "LogAI-CPP API",
  "version": "0.1.0",
  "status": "running"
}

Functionality

The full LogAI-CPP API normally includes these features:

  • Log parsing (Drain, Regex, CSV, JSON)
  • Feature extraction from log data
  • LogBERT vectorization
  • One-Class SVM for anomaly detection
  • DBSCAN clustering

Note: This is a simple health check server. For the full functionality, please check the repository documentation.
